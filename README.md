##AI Resource Management System

## 🚀 Overview
Veseda is an AI-powered system that helps organizations monitor and optimize electricity, water, and manpower usage.

---

## 💡 Problem
Companies waste resources without knowing where inefficiencies occur.

---

## 🧠 Solution
We provide:
- Centralized dashboard
- AI-based inefficiency detection
- Actionable optimization suggestions
- Apply suggestion feature with simulated savings

---

## ⚙️ Features
- Login page
- Dashboard overview
- Electricity / Water / Manpower modules
- AI insight system (rule-based)
- Apply suggestion simulation
- Efficiency score system

---

## 🧠 AI Logic
We use rule-based AI:
- Detect high usage patterns
- Identify inefficiencies
- Provide optimization suggestions
