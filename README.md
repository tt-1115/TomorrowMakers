🌟 Overview
EcoSmart is a complete resource management platform designed to help organizations monitor, optimize, and reduce their electricity and water consumption. The platform uses AI-driven insights to automatically schedule resource usage during off-peak hours, detect anomalies, and provide real-time savings calculations.

Key Statistics
⚡ Electricity: Reduces peak usage by up to 25%

💧 Water: Saves up to 33% through smart scheduling

💰 Cost Savings: Average $0.65/day** or **$237/year per building

🌱 Sustainability: 92/100 average sustainability score
